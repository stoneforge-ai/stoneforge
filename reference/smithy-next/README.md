# smithy-next Prototype

This directory is a product and UI/UX reference prototype for Stoneforge V2.

- It is not the active production app.
- Active V2 code belongs at the repository root under `apps/` and `packages/`.
- Use this directory to study flows, terminology, and interaction patterns when shaping V2.

Run it only when you need the prototype:

```bash
npm install
npm run dev
```
