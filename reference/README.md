# Reference Material

`reference/` holds code kept for historical, behavioral, and UI reference while Stoneforge V2 is built at the repository root.

- `v1/` is the archived Stoneforge V1 workspace.
- `smithy-next/` is the UI/UX prototype reference used during V2 planning.

The root workspace intentionally excludes this directory. Treat it as read-only unless a task explicitly asks you to work here.
